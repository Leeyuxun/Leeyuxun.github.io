DVWA File Upload

发表于 2019-11-20 更新于 2020-11-21 分类于 DVWA 阅读次数：

前言

文件上传漏洞通常是由于对上传文件的类型、内容没有进行严格的过滤、检查，使得攻击者可以通过上传木马获取服务器的webshell权限，文件上传漏洞带来的危害常常是毁灭性的。

文件上传漏洞的利用的条件

  1. 能够成功上传木马文件
  2. 上传文件必须能够被执行
  3. 上传文件的路径必须可知

函数介绍：

参数介绍：

参数 说明
$path 必需，规定要检查的路径，在Windows中，斜线（/）和反斜线（\）都可以用作目录分隔符。在其它环境下是斜线（/）
$suffix 可选，规定文件扩展名，如果文件有suffix，则不会输出这个扩展名

MEDIUM

分析

分析：

Medium的代码对上传文件的类型、大小做了限制，要求文件类型必须是jpeg或者png，大小不能超过 100000B（约为 97.6KB）；

BurpSuite抓包，更改content-type即可；

攻击测试

  1. 上传LOW的php文件，显示上传失败；

  2. BurpSuite抓包，将Content-Type改为image/png，重新发包，文件上传成功，可以使用蚁剑连接；

  3. 也可以通过**%00截断上传绕过**，

    在php版本小于 5.3.4 的服务器中，当magic_quote_gpc=off时，可以在文件名中使用%00截断，可以把上传文件命名为shell.php%00.png

HIGH

分析

分析：

  • strrpos()函数

    函数返回字符串find在另一字符串string中最后一次出现的位置，如果没有找到字符串则返回false，可选参数start规定在何处开始搜索

  • strtolower()函数：

    把字符串转换为小写

  • getimagesize()函数

    getimagesize()函数用于获取图像大小及相关信息，成功则返回一个数组，失败则返回FALSE并产生一条E_WARNING级的错误信息

限制：

  • 读取文件名中最后一个.后的字符串，通过文件名来限制文件类型因此要求上传文件名形式必须是*.jpg、*.jpeg 、*.png三者之一；
  • getimagesize()函数限制了上传文件的文件头必须为图像类型；

IMPOSSIBLE

分析

分析：

  • 使用 imagecreatefromjpeg 或 imagecreatefrompng 去掉了不属于图片的部分；
  • 将文件重命名为随机字符串；
  • 增加Anti-Token防止CSRF攻击；
